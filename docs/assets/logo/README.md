# Shipli Logo Assets

Official logo for Shipli - AI-powered store compliance & code quality audits for Flutter apps.

## Files

| File | Description | Use Case |
|------|-------------|----------|
| `shipli-logo.svg` | Full logo (512x512) | Website, documentation |
| `shipli-logo-512.png` | Full logo (512px) | Large displays |
| `shipli-logo-256.png` | Full logo (256px) | Medium displays |
| `shipli-logo-128.png` | Full logo (128px) | Small displays |
| `shipli-icon.svg` | Simplified icon (64x64) | Favicon, badges |
| `shipli-icon-64.png` | Icon (64px) | Small badges |
| `shipli-icon-32.png` | Icon (32px) | Favicon |

## Design Elements

- **Ship hull** - Streamlined shape representing deployment/shipping
- **Sail with code brackets** - `< >` cutouts tying to code/programming
- **Checkmark** - Quality approval, passing audits, ship-readiness
- **Blue gradient** - Sky-500 (#0EA5E9) to Sky-600 (#0284C7)

## Color Palette

| Color | Hex |

## Usage Guidelines

- Maintain clear space around the logo (at least 10% of logo height)
- Don't stretch or distort proportions
- Use appropriate size variant for context
- Light backgrounds recommended for full color version
